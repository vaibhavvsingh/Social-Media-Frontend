# Social Media App (In-Progress!!)

This is the frontend of the full stack Social Media App that I'm workin on.
Its a work-in-progess project.

## Technology Stack Used

In the project, I am using MongoDB as the database, Express for backend server and routing, React for frontend UI and logic, Node for backend logic (MERN).

### `npm start`

Run this command in the local clone repository of this project to run this project locally. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
